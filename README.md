# solidity+react+web3JS 实战项目

solidity合约部分上链生产合约abi后，启动react项目，利用web3.js实现交互。
实现功能：token授权，链上合约质押，提取，转账。

## remix+solidity
solidity代码编译部署在remix上执行，在eth sepolia测试网上部署
